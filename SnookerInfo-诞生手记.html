SnookerInfo 诞生手记

发表于 2020-12-27 | 分类于 Snooker |
| 字数统计: 3,371

这是我写在 B 站的一篇文章

缘起

记得很清楚，那是高中的一个暑假，我看了人生中第一场斯诺克电视直播，知道了还有一种“台球”叫斯诺克。当时在外婆家，和我的表兄弟一起欣赏着电视里那些神乎其技的进球，惊叹于那弹无虚发的连续进攻。这对于三个刚从街边一块钱一盘的烂球桌上结束熬战的我们来说，简直就是看到了神一般的存在。看完直播后，我们决定回到球桌大干一场，结果出门猛如虎，上桌二百五🤪。

那时的我，只是单纯地喜欢台球，谈不上热爱，慢慢地也淡忘了斯诺克。至于为什么会喜欢，可能因为我对有准度要求的游戏都比较感兴趣，正如我喜欢 cs 一样。

读大学后，除了玩着没落的 cs 和崛起的 dota，也偶尔和同学去校旁的球房切磋。在那见识到不少高手，意识到普通人也可以完成清台。回宿舍后我就开始上网查资料，学习一些台球的理论知识。学习过程中，知道了斯诺克界有着“四大天王”的说法，我回忆起那个暑假看的那场比赛，原来是火老师和亨皇！具体什么比赛已经记不清，只记得当时火老师势如破竹，亨皇神情有些恍惚，应该已经是后亨得利时代了。

那时经常会去我的台球网、斯诺克前线等一些网站和论坛看别人的帖子和评论，也开始研究一些斯诺克的比赛录像。后来在查找直播网站的过程中，收藏了一个的博客，博主经常更新近期比赛的秩序册和收看方法，感觉如获至宝。后来谷歌越来越远，又经常去一个叫 qqzhibo 的网站去看比赛。

斯诺克在中国的发展过程中，上海五星体育起了很大的传播作用，我当时看的最多的视频就是五星体育的，那两个解说真的是铁杆火吹！五星体育后来应该改名叫星期五台球了？

情深

随着 4g 网络的普及和直播的兴起，斯诺克也有了网络直播。看了 kk 直播后，才知道我经常访问的博客就是传说中的欧神所写。同期的还有章鱼直播，任浩江老师解说的也非常到位。

后来巨星买断了十年直播版权，简直令人悲喜交加。是一位哲人曾经说过：“巨星在线是一款闻名遐迩的垃圾软件”，虽然它很烂，但是聊胜于无。我主要是看欧神和任浩江的解说，风格各异，各有精彩。夜深人静的时候，更喜欢在欧神的直播间凑热闹。

在观看斯诺克比赛时，我对斯诺克的热爱又重新燃起了。曾经好几年都没用过微博，后来为了解斯诺克的动态，又回到了微博。欧神、叉神、还有曾经的碧老师等用户是我获取信息的主要来源，自己也曾写过几篇“记忆”系列的长文发在微博上。因为时间问题，后来断更了。

在网上冲浪的时候，就一直在找一个能满足我对斯诺克数据需求的 App，至少要能随时随地查看赛程、排名、比分。然而，移动互联网的大潮都退去了，这样的 App 还是没有，于是我决定自己写一个。写到这里，大家还不知道我是做什么工作的，其实我是做 iOS 开发的。

我去 WorldSnooker 官网找了一遍，没有提供公开的 api。在官网找到了一个可以联系的邮箱，我写了一封邮件，内容大意就是我是一个中国的斯诺克球迷，但是在中国访问官网特别慢，我想写一个 app 来服务中国的球迷，能不能提供相关的数据接口？这封邮件没有得到回应。

国外有个网站 snooker.org，数据非常全面，也提供了公开的 api 免费使用，我尝试去使用过，感觉接口相当地不友好，所以放弃了。

还有一个叫 CueTracker 的网站，数据之全，令人震惊。据我了解，这个网站是由一个人独立完成的，至今已经很多年了，连 snooker.org 都会引用它上面的数据。我对这个作者真的是佩服的很，凭热爱维护了这么多年。我曾在推上私信过网站作者，他说打算公开一些 api 给用户付费使用，但是只是短暂地开放了一段时间后，服务就关闭了，现在也没有再开放过。

星期五台球有很多数据，但是我都用不了，pass！

正苦恼之际，17 年底发现有个叫“深度斯诺克”的小程序，上面有一些可用的数据（貌似是星期五台球提供的数据服务？），通过抓包工具收集到接口后，我就开始利用业余时间制作 app 了。不久之后，app 就上线 App Store 了。在 App Store 发布 app 是要每年交“保护费”的，当时想着用 app 卖点小钱平衡一下支出，就设置了一个白菜价：6 人民币。当时上线后，发了条微博。“深度斯诺克”的账号持有人应该是看到了我的微博，很快就给 api 加了鉴权方式，我的 app 就拉取不到数据了。而这仅仅距离 app 上线才过去了两三天，我去苹果后台看了一下数据，有几个下载量。我寻思着，这样用别人的数据等于一直被人掐着脖子，我没有必要再花时间和精力去捣鼓别人的接口和数据了。在失落中，我把 app 下架了。当时有个人还在我的微博下嘲讽过我，应该就是“深度斯诺克”的账号持有人。这个小程序现在也没人更新和维护了，那个人好像是去做斯诺克主播了吧？

通过网络，我找到了一个叫 SportTrader 的网站，这是一家专门出售体育行业数据的服务商。我曾经联系过他们，给予的回复是，他们只给企业提供服务，回文中也没有报价，想必价格不菲。

一直以来，我都苦于没有数据支撑，数据之殇的路上我尝试了很多。现在国内有几个网站好像有蛮齐全的数据的，其中最全面的就是星期五台球了，他们的数据怎么来的，我不了解。如果有了解这方面的朋友，或者知道如何获取到数据，可以和我联系，感谢！

我一直都是在做 iOS 开发，不懂服务端，为了做一个斯诺克数据 app，我利用业余时间自学了 go 语言。经过一段时间的摸索和开发，SnookerClub 大概是在 19 年 10 月上线了 App Store 。一段时间后，经过现学现卖，同名小程序也上线了。小程序相当于是 app 的阉割版，当时是想通过小程序给 app 导流的。后来 app 经过几个版本的迭代后，最终拥有了一下几个功能模块：球坛、数据、发现、我的。具体功能这里不展开来讲了，这里我截几张图给大家看看：

发现模块是具有一定社交属性的，用户可以根据地图上可以找到的球房自发地创建一个相应的聊天室，聊天室给经常来这个球房的打球的人提供了一个线上交流的平台。设计初衷是想根据球迷们的使用场景，把用户下沉到线下场景，通过球房这个点把有关联的球友们聚集在一起。用户与用户之间也可以点对点实时聊天，可以查看对方的资料和文章。这些功能都是接入第三方服务实现的，超过一定使用量后是要付费的，价格不便宜。

虽然 SnookerClub 完成了，但我并不满意，它只实现了我预期设计的 30%。主要原因是：1、数据匮乏；2、第三方服务需要付费；3、单枪匹马，实力和精力不济。所以，我几乎没推广过这个 app。

19 年苹果的 WWDC 发布会之后，我就试了一下新的开发框架 SwiftUI，立刻喜欢上了。当时就想，不如借机去学学 SwiftUI，直接把 app 推倒了重新来过，也正好可以练手。入坑 SwiftUI 后，发现根本不足以在生产中实践，就这样一直搁置了这个事情，直至今日。

大概一年以前，发现国外有个名叫 World Snooker Online 上线了，它用的正是 snooker.org 的 api 服务，至于是不是我在官网看到的 api 接口，我就不清楚了。这个 app 的数据很全，UI 是典型的欧美审美风格，我很少打开，有兴趣的朋友可以去下载来看看，也许你会喜欢。

距离 World Snooker Online 上线后不久，世界斯诺克官网终于终于推出了自家的 app：WST，去 App Store 搜索“World Snooker Tour”就可以找到，个人觉得还没有网页版好用。

说回小程序 SnookerInfo，它就是 SnookerClub，只是改了个名字而已。一开始我起名 Club，是因为我想做的是一个交流平台，类似线上 Club 之意。最近我思考过之后，暂时不再考虑做 app 了，主要还是因为个人时间和精力不够，毕竟每天只有 24 小时，一个人的力量十分有限。也许将来的某一天，我会重整旗鼓，这是后话。

在我的小程序上线后不久，星期五台球终于也上线了小程序，数据很全。当然，SnookerInfo 我还是会继续维护下去，为了这份热爱和情怀，它能活多久，我不清楚，希望越久越好吧。

小程序上线后，在两三个斯诺克群里转发过，也发了几条微博，这是我唯一的推广了。当时也不想太多人来用，因为访问太多服务器可能承受不了，这些都是我个人支出的，没有盈利可以对冲。就这样佛系地过了一年多，现在有比赛的时候，竟然也有高达千人访问。没有比赛的时候，真的是无人问津。

时隔十三个月之后，也就是上周，我终于想起来我的小程序了。给它加了两个页面：破百、147数据统计页面和Ast 排名，然后提交了一个新版本。当然，我还加了几个广告位。为了不影响用户体验，我把这几个广告都加在了页面的最底部。这些广告都是系统根据用户喜好推荐的，时有时无。其实就我这点用户量，广告收益是杯水车薪，只是为了平衡服务器的支出而已。所以，恳请使用 SnookerInfo 的朋友们，如果你有幸看到了广告，麻烦点击一下，算是对的支持和鼓励，感谢！

现在的 SnookerInfo，至少能满足我作为一个球迷，对数据的基本需求：赛程、排名、比分。至于更多数据的查询，我一般会去上面提到的 snooker.org、CueTracker 或维基上查询。

短期内，我暂时不会去更新小程序的主要功能。如果你对 SnookerInfo 有什么好的建议和想法，可以留言或私信我。

厮守

正所谓“萝卜白菜，各有所爱”，我不知为何会喜欢斯诺克，但我知道它将会伴我一生。

很庆幸我们生在了这个时代，“四大天王”和“七五三杰”给我们带来了无数的动情时刻。

不知道以后还会不会有这样美好的时光，至少我们曾经拥有过，它已经深深地烙在了我们的记忆中。

题外话：

我是个内向的人，交际极少。一直以来，我都是被动输入型的人。直到 19 年的时候，我才接触到 B 站，它改变了我，使我乐于分享，使我变成了一个主动输出型的人。

我会在这里发一些自己剪辑的视频，也会搬运其它优质视频，有时会发一些斯诺克相关的动态和感悟。其实这是我的斯诺克“记忆”系列的延续，以前是文字，现在是视频。

感谢 B 站，我很喜欢这里的文化和氛围，希望它能越来越好。

“Happiness is only real when shared”，这句话很长时间一直压在我心底，分享给大家。

感谢你们聆听我的故事，很高兴认识你！

最后祝大家新年快乐，“牛”气冲天！
